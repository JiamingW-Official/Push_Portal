Rational Design

How We Make Staying On-Platform The Rational Choice

Five mechanisms that make the platform path obviously superior.

01
Future opportunities only exist on-platform
The next campaign, next high-paying merchant, next tier upgrade — all only through the platform. Bypassing today means missing tomorrow.
02
Private deals don't inherit platform credit
Creator Passport scores and merchant reliability ratings only exist on-platform. Off-platform deals start from zero every time.
03
On-platform re-booking is easier than DMs
One-click: duplicate campaign → invite same creator → auto-set payout → auto-collect proof → auto-settle. Why negotiate via DM?
duplicate → invite → payout → proof → settle — one click
04
High-tier benefits only settle on-platform
Premium payouts, priority matching, exclusive campaigns, bonus multipliers — all tied to platform transactions. Private deals can't unlock these.
05
Light rules, not heavy bans
Don't punish contact — incentivize transactions. Make the on-platform path so smooth that going off-platform feels like extra work.
