# Push Creator System

## Philosophy
"Anyone can start. Performance determines how far you go."
Creators are valued by completion rate, reliability, merchant satisfaction, and commercial results — not follower count.

## 6-Tier System v3.0

| Tier | Push Score | Compensation | Commission | Payout Speed | Concurrent Campaigns |
|------|-----------|-------------|------------|-------------|---------------------|
| Seed 🌱 | None (provisional 50) | Free product only | 0% | Instant redemption | 1 |
| Explorer 🔍 | 40+ | $10-15/campaign | 0% | T+3 | 2 |
| Operator ⚡ | 55+ | $15-25 + 3% commission | 3% | T+2 | 3 |
| Proven ✅ | 65+ | $25-40 + 5% + 5% bonus | 5% | T+1 | 4 |
| Closer 🎯 | 78+ | $40-75 + 7% + 10% bonus | 7% | Same-day | 6 |
| Partner 👑 | 88+ | $75-150 + 10% + 15% bonus | 10% | Instant | 8 |

### Seed Tier — Key Differentiator
- **Zero barrier:** No follower count requirement. Just register + fill profile.
- **15-20 slots per campaign** (high volume, low cost) — transforms Push from "influencer platform" to "local community acquisition platform"
- **Upgrade path:** Complete 3 campaigns + provisional score ≥ 40 → auto-promote to Explorer

### Progression Philosophy
Each tier unlocks exactly what the previous tier's creators start craving:
- Seed → Explorer: Official identity + cash payment
- Explorer → Operator: Commission = passive income (behavioral inflection point)
- Operator → Proven: Premium brand partnerships
- Proven → Closer: Dedicated manager + campaign co-design rights
- Closer → Partner: Co-branding + platform decision-making power

### Merchant Tier Linkage
- **Starter ($19.99/mo):** Seed + Explorer + Operator, 3 slots/campaign
- **Growth ($69/mo):** All tiers (Proven priority), 5 slots/campaign
- **Pro ($199/mo):** All tiers + can invite specific Closer/Partner, 8 slots/campaign

## Commission Structure
- 30-day attribution window
- Commission from every transaction via creator's referral link/QR code
- Operator tier (3%) = first unlock → behavioral inflection point
- Progressive scale: 3% → 5% → 7% → 10%
